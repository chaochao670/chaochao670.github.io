GVIM设置方法

Posted on 2018-05-16 | In 技术 | 浏览 次

gvim设置方法
set nu
set tabstop=4
set softtabstop=4
set shiftwidth=4 (tab缩进为4个空格)
colorscheme slate
set guifont=Courier_New:h19
set nobackup
set nowrap
set hlsearch (linux 下高亮显示搜索)
set fileencodings=ucs-bom,utf-8,cp936,gb18030,big5,euc-jp,euc-kr,latin1
set ignorecase 不区分大小写
小技巧:替换掉所有^M %s/\r/\r/g
删除所有空行 g/^$/d
删除回车 a,bs/\n// 每替换一次回车少一行
将单词或句子存入命名的缓存区
“ayy 存一行到a缓冲区
“ayw 存一个单词到a缓冲区
“ap 将a缓冲区内的字符粘贴出来
:%s/^(.)(\n\1)+$/\1/ 消除重复行
:%s/,/\r/g 将逗号替换为回车
GVIM通过运行vim脚本，进行多次指定替换
通过:Te打开多个标签页，下次想直接打开这么多指定标签页，就用GVIM上面的保存会话，保存为session.vim文件，下次加载会话即可
:s/^/\=range(5, 10) 在光标处生成5-10一行一个数字
:Te像chrome那样多标签页打开，ctrl+page down可以切换标签,:tabnew新标签
“bp 将b缓冲区内的字符粘贴出来
ctrl+L整行补齐,ctrl+p关键字补齐
=%缩进函数内字符，gg=G缩进整个文档
ctrl+o向后追赶鼠标历史位置，ctrl+i向前追赶鼠标历史位置
将gvim.exe放入path后，gvim -p .java可以多标签页打开
:mks a.vim可以将当前多标签状态和标记记录为sesson文件，下次用命令gvim -S a.vim可以直接恢复
m[a-zA-Z]大写标记是全局的，多文件中唯一的，小写标记是单文件内的
安装exuberant ctags，地址：http://ctags.sourceforge.net/index.html
下载的文件名为ctags58.zip，解压后，将目录中的ctags.exe复制到目录D:/Program Files/Vim/vim72。
将ctags.exe的运行路径G:\vim\vim74加入到PATH，在VIM启动设置中加入
set tags=tags
set autochdir
打开cmd命令行，切换到你要查看的源代码的根目录处，运行
ctags -R
将会在此目录处生成一个tags文件。 用gvim打开一个代码文件，将光标放到某一函数名上，如下图的UpdateViewByPosNo()，按下”ctrl+]”，光标会自动跳转到定义处。
按下”ctrl+T”会跳回到原来的位置。 变量、结构体、宏等等，都可以这样做。 当你的源文件有更新时，只能重新运行ctags -R命令，来更新tags文件。
下载安装taglist
http://www.vim.org/scripts/download_script.php?src_id=7701
将taglist_45.zip解压，解压后包含一个doc文件夹和一个plugin文件夹，将其中内容分别复制到C:\Program Files\Vim\vim72下的doc及plugin中。
安装完成后在VIM中输入:Tlist,成功的话会显示Tlist
安装windowManager http://www.vim.org/scripts/download_script.php?src_id=754
用相同的方法将winmanager.zip解压和拷贝，在_vimrc文件中加入以下内容：
let g:winManagerWindowLayout=’FileExplorer|TagList’
nmap wm :WMToggle
用gvim打开代码文件，normal状态下输入命令”wm”，窗口如下
其中左上是netrw窗口（浏览文件），左下是TagList窗口，再次输入”wm”时这两个窗口会关闭。ctrl+w 加上hjkl可以在多窗口间切换
安装grep.vim http://www.vim.org/scripts/download_script.php?src_id=7645 下载后移动到plugins里，在启动设置里添加
nnoremap :Grep
用gvim打开源码文件，并将光标定位到要查找的内容上，按下F3，确定要查找的内容和搜索范围，gvim会在弹出的QuickFix窗口中列出所有符合条件的搜索结果.
vimgrep不好用,用nodepad的功能代替
linux中添加标签匹配功能
sudo vim /etc/vim/vimrc
使用浏览树功能 解压出来放入plugin 然后:NERDTree ctr+w+h 光标focus左侧树形目录,修改/etc/vim/vimrc map :NERDTreeToggle 即可使用快捷键打开和关闭树目录
