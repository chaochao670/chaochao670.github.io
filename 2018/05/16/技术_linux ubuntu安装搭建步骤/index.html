linux ubuntu安装搭建步骤

Posted on 2018-05-16 | In 技术

16.04
1.将16.04的镜像通过大白菜ISO模式写入U盘，重启电脑 使用ubuntu而不安装
桌面上的install 不连wifi 其它选项（自己分区，预先分出来一个UBUNTU的空闲区）
空闲区 -挂空 +修改 改变 按照下列顺序来
1 逻辑分区 200M 起始 Ext4 /boot
2 逻辑分区 4000M 起始 交换分区 无挂载点
3 主分区 15000M 起始 Ext4 /
4 逻辑分区 剩余全部 起始 Ext4 /home
引导分区设置为/boot所在分区号
安装后重启会进入windows,用360安装easyBCD,添加引导入口 选择linux标签页 名称输入ubuntu 16.04 Drive选择/boot的200m，添加即可，至此安装成功

二.安装输入法
sudo apt-get install ibus ibus-table ibus-table-wubi
安装五笔输入法， 系统设置-语言支持-输入法系统-ibus 然后 ibus-setup 添加极点输入法,在文本输入设置中添加 极点五笔输入即可
vim设置搜索高亮 set hlsearch

解决鼠标闪烁问题 未知显示器设置为“关闭”状态
安装flash看视频 扳手 软件 其它软件 Canonical 合作伙伴 sudo apt install adobe-flashplugin
安装字体 将windows下的字体复制到/usr/share/fonts/truetype/mywinfonts下，
sudo mkfontscale
sudo mkfontdir
sudo fc-cache -fv 即可安装成功 安装字体firefox浏览器显示有问题用 fc-list :lang=zh-cn 查看支持的字体
安装chrome浏览器
sudo wget http://www.linuxidc.com/files/repo/google-chrome.list -P /etc/apt/sources.list.d/
wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add - #本句没有执行成功依然安装成功
sudo apt-get update
sudo apt-get install google-chrome-stable
sudo apt-get install rar #安装rar解压程序 rar x a.rar 解压文件
sudo apt-get install pinta #安装简单:图片编辑工具
安装notepad
sudo add-apt-repository ppa:notepadqq-team/notepadqq
sudo apt-get update
sudo apt-get install notepadqq
