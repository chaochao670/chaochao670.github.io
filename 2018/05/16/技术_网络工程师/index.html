二.IP地址:
a类地址，0开头，1+3字节的表示方法。
b类地址，10开头，2+2字节的地址

三.IP地址优化历程
为了解决二级IP地址利用率低且给每个网络号分配一个转发规则路由表会很庞大的问题，同时两级IP地址不够灵活，推出了三级IP地址
IP地址 ::= { <网络号>, <子网号>, <主机号>}
子网划分是单位内部的事，外界看不到。但单位的路由器收到数据时要知道应该转发给哪个子网，就要用子网掩码。
子网划分时，路由表必须包含以下三项内容：子网掩码、目的网络地址和下一跳地址
从数据包中取出目的IP地址D。
目的地址D与各个子网掩码相与，如果结果N等于路由表中的目的网络地址，则直接从路由表中记录的对应下一跳地址转发出去(要变换目的物理地址再发送)，至此直接交付任务完成，否则向下判断。
如果路由表中有地址为D的特定主机路由，则直接交付。
如果路由表中有能够到达D的路由，把数据转给此路由，否则继续下一步。
上述匹配均失败则转发给默认路由器，否则执行6.
报告转发分组出错。
无分类域间路由CIDR(Classless Inter-Domain Routing)是对三级IP地址的又一改进

四.打印机共享三种方法
1.打印机通过串口接在有队列打印服务程序或打印共享管理软件的PC上，网络其它成员通过网络共享这台打印机。
2.单立式专用打印服务器，用单立式打印服务器代替上面普通PC,此服务器将网络接口转换成并口再连接打印机，并口传输速度为170kbps~1mbps。
3.网络打印机，有EIO或MIO插槽的打印机安装内置式打印服务器(相当于集成了网络打印功能的网卡),适合于大型公司的办公网络。

一台HP LaserJet 4100可为整个网络提供相当于十几台普通打印机的容量和性能，价格相当于普通打印机的2~3倍。
打印机参数
DPI:每英寸上可打印的点数
PPM:papers per minute每分钟输出的纸张数
连续打印能力：连续打印超过上百张打印速度明显降低，必须停机降温一段时间。
五.网络需求分析，系统方案设计，施工方案设计。
需求分析一般由经验丰富的系统分析员来完成。
系统集成商的费用来自硬件差价，系统集成费，软件开发费用三部分。
网络数据有三种类型：OA/WEB类应用，数据交换频繁但负载小.二.ftp文档文件传输，数据发生不多，负载较大，容许延迟.三.流式文件，rm/vod数据随时发生负载巨大，需要图像声音同步。
施工方案设计，由用户提供建筑工程图，了解中心机房位置，信息点数，信息点与中心机房的最远距离，分析施工难易程度。
服务器接入：并行快速以太网冗余接入，两个100mbps服务器专用高速以太网卡分别接入两台交换机中，此种方案比较流行。
1、配置路由器R提供DHCP服务
R(config)#service dhcp //开启DHCP 服务
R(config)#no ip dhcp conflict logging //关闭DHCP 冲突日志
R(config)#ip dhcp pool ccie //定义地址池
R(dhcp-config)#network 192.168.1.0 /24 //DHCP 服务器要分配的网络和掩码
R(dhcp-config)#domain-name cisco.com //域名
R(dhcp-config)#default-router 192.168.1.1
//默认网关，这个地址要和相应网络所连接的路由器的以太口地址相同。
R(dhcp-config)#netbios-name-server 192.168.1.2 //WINS 服务器
R(dhcp-config)#dns-server 192.168.1.4 //DNS 服务器
R(dhcp-config)#option 150 ip 192.168.1.3 //TFTP 服务器
R(dhcp-config)#lease infinite //定义租期
R(config)#ip dhcp excluded-address 192.168.1.1 192.168.1.5 //排除的地址段
