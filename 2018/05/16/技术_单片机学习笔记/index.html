单片机学习笔记

Posted on 2018-05-16 | In 技术 | 浏览 次

基础知识
STC 89C 52RC 40C-PDIP
52的2是表示内存大小，51为1X4K=4K的内存52为2X4K=8K 40表示支持最大的晶振为40M
Keil C51是51系列兼容单片机C语言软件开发系统
SDC-ISP 串口下载程序，选择单片机型号，选择COM端口，先点击下载，再打开单片机(冷启动)
发光二极管 点亮的电流约为 电流3ma ，压降为1.7V，据此计算串联的电阻(一般串联1K欧电阻)
电阻分类(上拉电阻，下拉电阻，限流电阻)
开关电源 电流过大断开 电流小自动打开(usb供电即是)
要停止调试，先按单片机上的复位按钮，再停止调试(51单片机)
时钟周期(振荡周期),,状态周期(是时钟周期的两倍)，机器周期(一个机器周期有6个状态周期，完成一个独立的操作，为12个时钟周期)，指令周期(由一个或几个机器周期完成一条指令).
C语言编程中会屏蔽这些周期不用考虑，可用keil编译器设置晶振软件模拟有运行时间，软件仿延迟时间，更精确的延时需要有定时器
继电器从电源到地要反向接一个二极管，继电器断开时电磁感应会产生一个反向的电压，该二极管允许反向电压的流过，不至于下面的高电压击坏某些器件如三极管。
引脚功能的复用，由寄存器的值控制何时使用第一个，何时使用第二个。
计数器 引脚符号为T0 T1,可以计数来了多少个方波，如果是正弦波可以用比较器变成方波然后输入计数器引脚。
复位引脚外接的电容作用为上电自动复位，上电电容充电，复位引脚拥有电压就复位，充满电后放电。
地址可以被8整除的可以直接进行位操作。
时钟源的晶振，11.0592MHZ配30P的电容，6M晶振配20P电容。
内部有上接电阻的准双向I/O口，要写的时候先向其写个1，作输入口使用时，要先向该口写1(要准备一下才能输入用)
外部上拉电阻的原因是因为单片机的输出电流太小，无法带动二极管等发亮(一般P0口内部没有上拉电阻)，上拉电阻(一般选用10k)接电源可以加大负载电流。
每个char型变量会占用随机存储器RAM，unsigned char code c1;//加上code变量会存入ROM(程序区)，相应如果写data是写入RAM(代码区)的意思.
中断过程: 中断发生->中断响应(停下手中A活)->中断服务(处理事件B)->中断返回(继续处理A)
优点是分时处理多个I/O中断，不同优先级可产生嵌套中断。
中断(电平触发和边沿触发)
EA=1;//开总中断 EX0=1;//允许0号中断
中断函数定义(不需要声明) void exter0 () interrupt 0 //0表示第0号中断,后面添加using 1表示使用第几组寄存器
{//计数器溢出变为0，程序重新装一次值
}
定时器
软件定时:delay 或者a–等，降低了CPU利用率
时基电路定时：外接必要的器件，定时值和范围不由软件控制
TMOD是定时器工作方式寄存器，确定工作方式和功能，计算初值并写入TL0和TH0

    Gate是门控位，0时TR0或TR1置1即开始计数；1时TR0或TR1为1而且外部中断也为高时才开始计数(多了一个条件)
    C/T是：定时计数模式选择，T(0)是定时，C(1)是计数
    M0M1指定四种工作方式，分别是 13位、16位 定时/计数器，8位自动重装和两个独立的计数器
    计算方法:要获得50ms的定时，16位寄存器用(65536-50000)/256得高八位的计数初值，(65536-50000)%256得低八位初值
    从此16位初始值计50000nm即是50ms的定时器
TCON是控制寄存器，控制T0,T1的启动和停止，(TR0和TR1复位)设置溢出标志，中断方式时，对IE赋值开放中断
计数脉冲来源:1.来自系统时钟12分频后送来(此时作为定时器)，2.T0或T1引脚输入的外部脉冲源(此时作为计数器)，计数到达时计数器的溢出会使TCON的TF0或TF1置1，发出中断请求

A/D转换
A/D转换需要时间，不同data sheet上面标明了转换需要的最小时间，18f452转换需要的最小时间是Tad>1.6us,
根据单片机频率设置时钟转换频率，使时钟转换频率足够慢，能够正确完成转换。如以2Tosc作为AD转换时钟周期，
当单片机主频为1.25MHZ或更低时，可以正常完成转换。
A/D转换配置 C/R模拟输入通道编号/A/D转换通道编号 1/0表示1号模拟输入即AN0
右边的0表示用的标准电压作参考电压，用一个外部输入作为参考电压则为1，两个则为2
PWM
如在正弦波中正半轴才能正常工作的机器，冲量相等的脉冲加在有惯性的环节上其效果基本相同
平均输出电压决定了电机的转速，其工作的平均电压为Ua=(Ton/T)*U=pU( p是密度的rou,表示占空比)
调节输出电压平均值Ua，可以调Ton—PWM(脉冲宽度高度)，可调T-脉冲频率调制(PFM)
振荡器RC外部电阻/电容振荡，OSC2被配置为RA6,使用IO引脚即为使用内部晶振提供时钟频率，可以节省IO引脚,否则IO引脚必须作为外部晶振的输入

18f452，pwm周期=(pr2+1)4ToscTmr2预分频值，data sheet给出的占空比=(ccpr1:ccp1con<5:4>)Tosc*Tmr2预分频值，是正脉冲的时间，正确的占空比是个比值，由上公式算出ccpr1l/pr2接近于正确占空比。

异步通信
空闲一直是高电平，起始位是一个低位，结束位是两个高电平。一个字节先发送低四位，低电平是0，高电平是1。
接收到规定的数据位个数和校验位后,通信接口电路希望收到停止位S(逻辑1)，若此时未收到逻辑1，说明出现了错误，在状态寄存器中置“帧错误”标志，有时波特率不对会产生帧错误。
